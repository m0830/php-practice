# PHP練習問題② 説明

## 下記for文の`$i = 0`, `$i <= 4`, `$i++`, `echo $i`がそれぞれ、どの順番で処理されるか、また、何をしているのかを説明してください。

```
for ($i = 0; $i <= 4; $i++) {
    echo $i;
}
```

1. $i = 0：0からスタート
2. $i <= 4：4より小さい間、繰り返す
3. echo $i：$iを表示
4. $i++：中身を1足してく

## for文とforeachはどのように使い分けるのか説明してください。
- 指定した数分繰り返し処理を書くときに使う構文がfor文で、配列又は連想配列から要素を1つずつ取り出してコレクションの要素の数分だけ繰り返し実行するのがforeach
